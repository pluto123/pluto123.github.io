三種常用的 Hook

  1. State Hook : React.useState()
    此 hook 可以讓我們的函數式組件使用 state
    使用 React.setState(初始狀態) 初始化狀態並且會兩個元素的數組，第一個元素是當前狀態，第二個元素是更新狀態的函數．
    而返回的更新狀態函數，也有兩種方式，一種是傳數值直接更新，另一種是傳函數．
    當頁面上有多個狀態就需要呼叫多次 React.setState(初始狀態) 將狀態一一初始化並取得數組．

  2. Effect Hook : React.useEffect()
    此 hook 可以讓我們的函數式組件使用 react 生命週期回調
    使用類別組件在組件一掛載後設定定時器自動累加 count，並且在卸載組件時清除定時器

    而要在函數式組件中實作的話，則需要借助 React.useEffect()

  3. Ref Hook : React.useRef()
    此 Hook 可以讓我們的函數式組件取得標籤對象的參考，進而儲存或取得數值
