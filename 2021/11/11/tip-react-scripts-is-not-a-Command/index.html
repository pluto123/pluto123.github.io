Tip - 'react-scripts' 不是內部或外部命令、可執行的程式或批次檔。

Posted on 2021-11-11 In Tip
Symbols count in article: 162 Reading time ≈ 1 mins.

當我們由 Git 庫下載 React 源碼後，第一次啟動npm start會發現以下錯誤 :

這是因為一般上傳 React 專案時，是不會連該專案所安裝的模組 (下圖黃色框部分) 都上傳，而只是上傳該專案所用的模組設定檔(package.json下圖紅色框部分)，所以我們需要在重新安裝一下。

解決方式 : 在終端機上輸入npm install進行安裝。
