查詢系統 Java 版本

在 Ubuntu Linux 系統上我們可以同時安裝多種 Java 版本，可以透過 update-alternatives 這個工具來設定系統預設的 Java 版本。
首先用 --query 或是 --display 列出目前系統預設的 Java 版本以及所有可用的 java 版本。
