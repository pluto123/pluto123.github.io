    所以我們可針對這些錯誤進行改善，增加使用者體驗，底下就是一個最簡單的方式，讓所有的值都有定義 : onst {id, title} = this.props.location.state || {}這種寫法是當this.props.location.state是 undefined 時就拿後面的{}進行取值．
