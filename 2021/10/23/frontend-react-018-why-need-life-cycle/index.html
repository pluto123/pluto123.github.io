結果會看到計數器的值不會是每秒累加一，這是因為每次呼叫 setState()後就會呼叫一次render()，而呼叫一次render()就會創建一個定時器，所以到最後創建多個定時器同時對 state 進行更新。

所以希望 React 可以在一個時間點，即在頁面掛載時，調用一次設定定時器的函數。
React 提供一個回調函數componentDidMount()，在頁面掛載完成後進行呼叫．
