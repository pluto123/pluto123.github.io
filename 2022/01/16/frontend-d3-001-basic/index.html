D3 - 001 SVG 簡介

Posted on 2022-01-16 Edited on 2022-01-25 In D3
Symbols count in article: 5.5k Reading time ≈ 5 mins.

簡介

Scalable Vetor Graphics (SVG) 是個向量圖形格式，用於建立二維圖形，也支援互動操作與動畫．SVG 檔案是基於 XML 的文字檔案．SVG 產生的是 DOM 樹．

概述

SVG 定義一系列可渲染的圖形元素，像是基本的形狀、文字元素及直線與曲線．這些元素的外觀可以透過外觀屬性加以控制，像是尺寸、位置等．
SVG 也定義一些結構化的元素，除了可被組織在一個 SVG 檔案中，也可以把其他元素組合在一起，進行轉換或重用．
SVG 的渲染是依據文件的先後順序的，所以在文件前面的元素會被後面的元素壓在底下．
SVG 使用圖形座標，原點在圖的左上角．
SVG 可以接收使用者事件，然後呼叫對應的事件處理程式．

形狀

每個形狀都有自己的屬性組可以控制其大小與位置．

tag attribute description
<rect> x, y 左上角的座標
width, height 寬高
rx, ry 邊角半徑
<circle> cx, cy 中心座標
r 半徑
<ellipse> cx, cy 中心座標
rx, ry 水平與垂直半徑
<line> x1, y1 開始點
x2, y2 結束點

路徑

元素只有一個屬性 d，這個屬性 d 是由命令與座標結合的字串，中間以空白作為分隔符．所以有命令皆是以一個字母構成，命令如果是大寫字母，則表示之後的座標是絕對座標，如果是小寫字母，則表示之後的是相對座標．

文字

可使用 <text> 元素將文字包含在 SVG 圖形中，可以內嵌 <tspan> 元素，它用於在原本字串中定義出子字串

只有 <text> 中才有 text-anchor 屬性，是控制對齊方式

<textPath> 可以沿著指定的 <path> 上顯示文字

文字字體的顏色是由 fill 屬性所控制，而 stroke 則是在字型的外圍有一個框線顏色，只有在字體較大時才明顯
<text> 並不會有文字換行機制，多行文字必須由多個獨立的 <text> 組合而成．

外觀顯示屬性

屬性 說明
stroke 元素外框線顏色，預設為 none
stroke-width 元素外框線寬度，預設為 1
stroke-opacity 元素外框線的透明度，0 為完全透明，預設為 1
stroke-dasharray 元素外框線的破折線和差距樣式
fill 元素內要填充的顏色，預設為黑色
fill-opacity 元素內不透明度，0 為完全透明，預設為 1
font-family 字體家族
font-size 字體大小
font-style 字體樣式
font-weight 字體粗細
cursor 當滑鼠游標放在元素上方時要顯示的游標形狀
opacity 控制元素的透明度
display 控制元素顯示與否，不可被繼承，當設為 none 時，接收不到事件
visibility 控制元素是否可見，可以被繼承，當設為 hidden 或 collapse 時，也會接收到事件

轉換

使用 <transform> 進行元素轉換

操作 說明
translate(dx, dy) 把目前元素平移
rotate(phi, x, y) 以座標 x, y 旋轉一個角度 phi
scale(fx, fy) 縮放
skewX(phi) 沿著水平推移轉換
skewY(phi) 沿著垂直推移轉換
matrix(a, b, c, d, e, f) 座標轉換

元素 說明

<use> 可參考到另一個元素
