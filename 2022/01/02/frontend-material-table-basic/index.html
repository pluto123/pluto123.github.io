Sort

原本預設是每個欄位都可以進行排序，藉由設定 options 中的 sorting 值可以關閉欄位名稱旁的排序功能

若只要關閉特定欄位的排序功能，則在該欄位的 sorting 設為 false

若要設定預設排序方向時，用 defaultSort 設定

客製化排序，use customSort

code

Search

關閉 search 功能

原本在表的右上角有個 search 欄位消失

使用 searchTest 設定預設搜尋文本

指定不搜尋某個欄位，使用 searchable

使用 searchFieldAlignment 調整搜尋欄位位置

使用 searchAutoFocus 載入頁面後 search 欄位自動取得焦點

使用 searchFieldVariant 設定搜尋框樣式
searchFieldVariant: [standard | filled | outlined]

code

Filter

使用 filtering 將過濾功能打開

欲關閉某欄位過濾功能，則使用 filtering 關閉該欄位的過濾功能

使用 filterPlaceholder 設定提示

code

Pagination

使用 paging 關閉分頁功能

使用 pageSizeOptions 設定一頁顯示的資料數

使用 pageSize 設定一頁預設顯示的資料數

使用 paginationType 改變換頁方式
paginationType: [normal | stepped]

使用 showFirstLastPageButtons 開關跳至第一頁及跳至最後一頁的按鈕

使用 paginationPosition 改變分頁按鈕位置
paginationPosition: [top | bottom | both]

code

全部資料被匯出

Selection

打開選擇按鈕讓使用者可勾選資料

disable 全選的 checkbox

如下圖所示，當我們選擇了某幾個資料時，將會有一個紅色提示框顯示我們選擇了幾列，同時在 Search 右邊的 export 和 add 圖示按鈕也消失了

如果我們要消除該提示的話，要設定 showTextRowsSelected

如果有設定 selection 為 true，則客製化 action button 中的 isFreeAction 設定為 false，也不會出現在每個列中，而當有資料被選擇時才會出現

沒有選擇資料時，客製化 action button 不出現

有選擇資料時，客製化 action button 出現

而當我們設定 selection 為 true，則客製化 action button 中的 isFreeAction 設定為 true，也不會出現在每個列中，但是會出現在 search 框的右邊，而當有資料被選擇時就消失掉

沒有選擇資料時，客製化 action button 出現

有選擇資料時，客製化 action button 消失

Selection isFreeAction description
false false 在每列中會有 action button
false true 在搜尋框旁邊
true false 沒有勾選資料時，不出現；勾選資料後，出現在搜尋框旁邊
true true 沒有勾選資料時，出現；勾選資料後，不出現

監控當有資料被勾選或取消時的觸發事件
