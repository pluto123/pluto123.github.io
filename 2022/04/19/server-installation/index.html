Dell T640 Server Installation

Posted on 2022-04-19 In Server
Symbols count in article: 432 Reading time ≈ 1 mins.

安裝前準備

  • 透過 Controller Port 連線
    將網路線連接背板上的網路孔 (下圖紅色框)，接上螢幕看是被配置哪個 IP 並記下
  • 透過瀏覽器登入
  • 登入後點選虛擬控制台
  • 虛擬控制台畫面選擇進入 BIOS

設定 RAID

  1. 選擇 Device Settings 以設定 RAID
  2. 選擇 RAID Controller

刪除虛擬硬碟

  1. 選擇 Virtual Disk Management
  2. 選擇要刪掉的虛擬硬碟
  3. 選擇要執行的動作
  4. 按下 Go 執行
  5. Confirm
  6. 執行完成
  7. 執行結果 - 減少了一個虛擬硬碟

Convert to Non-RAID Disk

  1. 選擇 Configuration Management
  2. Convert to Non-RAID Disk
  3. 選擇要還原的硬碟
  4. 確認執行
  5. 執行成功

建立虛擬硬碟

  1. 選擇 Create Virtual Disk
  2. 設定 RAID 並選擇實體硬碟
  3. 選擇那些實體硬碟要組成一顆虛擬硬碟
  4. 確認後執行
  5. 選擇實體硬碟類型與要設定的實體硬碟
  6. 執行成功
  7. 選擇 Create Virtual Disk
  8. 確認後執行
  9. 執行成功
  10. 一路按 back 回到首頁按下 Finish
